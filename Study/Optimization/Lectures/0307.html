Grid Search under convex assumption : Narrowing the intervals!

  1. \(x^*\in[a,b]\)

  2. Have to find \(c,d\) such that \(x^*\in[c,d]\subset[a,b]\).

  3. Repeat : Find \(c<d<e\) such that \(f(c)>f(d)\) and \(f(d)<f(e)\).

Example

\(f(x)=e^{-x}+x^2\)

Step.1 : Set an initial solution \([x_{old},y_{old}]\)

Step.2 : Take \(\{z_1,\cdots,z_n\}\) such that \(x_o<z_1<\cdots<z_{n}<y_o\)

Step.3 : Calculate \(f(z_1),\cdots,f(z_n)\)

Step.4 : Find \(i^*=\arg\min_{i\in\{1,\cdots,n\}}f(z_i)\)

Step.5 : Update \(x_{new}=z_{i^*-1}\) and \(y_{new}=z_{i^*+1}\).

Repeat Step.2 ~ Step.5 until convergence with tolerance level \(\varepsilon\).

rm(list = ls())
par(mar = c(5, 5, 2, 2))
f <- function(x){exp(-x) + x^2}

# Initial bounds
a <- -4 ; b <- 10 ; n <- 10 ; eps <- 1e-6
iter.max <- 1e2

plot(function(x)f(x), a, b, ylab = expression(f(x)))
text(2, 20, expression(paste(f(x), '=', e^{-x} + x^2)))

# Step.2 ~ 5
trace.v <- rep(NA, iter.max)
for (j in 1:iter.max){
  print(j)
  trace.v[j] <- f(a)
  x.v <- seq(a, b, length.out = n)
  f.v <- rep(NA, n)
  for (i in 1:n) {
  f.v[i] <- f(x.v[i])
  }
  opt <- which.min(f.v) # i*
  a <- x.v[opt - 1]
  b <- x.v[opt + 1]
  if (b - a < eps) break
}
